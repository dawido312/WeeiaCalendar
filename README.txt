Sposób użycia:
localhost:8080/getweeiacalendar?year=[rok]&month=[miesiąc]

Zapytanie generuje kalendarz w formacie .ics dla podanego miesiąca oraz roku.
Parametrami domyślnymi są year=2019 oraz month=11.

Przykładowe zapytania:
localhost:8080/getweeiacalendar?year=2019&month=12
localhost:8080/getweeiacalendar
